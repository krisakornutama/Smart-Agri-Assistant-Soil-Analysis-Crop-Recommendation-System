# Smart-Agri-Assistant-Soil-Analysis-Crop-Recommendation-System
Farmers in Phang Khon lack soil tools, causing high costs. This study built a portable AI prototype using a 7-in-1 sensor and ESP32 with an MLP model. To fix low accuracy, a Hybrid Logic system was added via a Web App. This provides reliable, real-time crop advice, helping farmers make better decisions directly in the field.

Thai agriculture is facing challenges such as high costs, soil degradation, and unpredictable weather patterns, making experience-based farming insufficient. One of the most crucial pieces of information is soil quality. However, standard soil testing is cumbersome, time-consuming, and expensive, making accurate soil analysis inaccessible to most small-scale farmers in Phangkone District, Sakon Nakhon Province.

The soil in this area is predominantly sandy and sandy loam, suitable only for certain crops. Without accurate soil data, farmers may use fertilizers and water inappropriately, leading to wasted resources and low yields. Furthermore, a significant problem is the lack of open digital soil datasets (raw and labeled datasets) that developers and researchers can use to create AI systems to assist farmers.

Therefore, this project aims to develop a portable soil quality analysis and crop recommendation device using artificial intelligence. This device will measure key soil parameters and process them using an MLP (Multi-Level Processing) model to instantly recommend suitable crops. This will reduce costs, increase efficiency, and make access to soil data easy and affordable for small-scale farmers, as well as fill data gaps for future research.
